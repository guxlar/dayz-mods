[ES]

Construcciones para mercado en el mapa Deer Isle, a ser usadas por dueños de servidores. Los objetos están en formato .dze (compatibles con el DayZ Editor) y usan elementos vanilla.

- deerisle-waldoboro-puerto-1 (playa al sur de Waldoboro)

INSTALACION (SOLAMENTE EN EL SERVIDOR)
- Mods requeridos en el servidor: @BuilderItems, @DayZ Editor Loader
- Los archivos están en la carpeta 'mappings' de este mod.
- Copiar los archivos .dze a la carpeta mpmissions/dayzOffline.Arland/EditorFiles

[EN]

Buildings for traders mapped on the Deer Isle map, to be used by server owners. Objects binarized in .dze format (compatible with the DayZ Editor), made of vanilla items.

- deerisle-waldoboro-puerto-1 (southern beach from Waldoboro)

INSTALLATION (ONLY ON THE SERVER)
- Mods required on the server: @BuilderItems, @DayZ Editor Loader
- The files are in this mod's folder 'mappings'.
- Copy the .dze files to folder mpmissions/dayzOffline.Arland/EditorFiles
